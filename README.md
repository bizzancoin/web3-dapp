# 介绍：

一款基于Matic 的NFT 游戏投资项目，包含推荐奖励模式。

用户投资USDT ，根据既定的条件规则，获取投资收益，投资收益直接返还到客户的TP或者imtoken 钱包地址，到帐USDT 或者代币。

| 模块         | 备注                 |
| :----------- | -------------------- |
| **智能合约** | Solidity             |
| **主链**     | Polygon，前身是Matic |
| **管理后台** | PHP 开发             |
| **前端web**  | VUE 开发             |

# 源码包含：

1. 智能合约.sol 代码；
2. 前端vue 源码；
3. 后端管理台 PHP源码；
4. 搭建、安装说明文档；
